Expand description

Instead of Base Actions, we can use a list of Functions to apply. Fun holds the implementation of all available functions.

Variants§

§

MvMain(i32)

Moves the main tape head by a (neg or pos) amount of cells.

§

MvWork(i32)

Moves the work tape head by a (neg or pos) amount of cells.

§

WriteMain(Gamma)

Writes the given letter from Gamma to the main tape at the current head position.

§

WriteWork(Gamma)

Writes the given letter from Gamma to the work tape at the current head position.

§

Geq()

Compares the next two letters A and B on the main tape, if A >= B, puts 1 in the third position, otherwise puts 0. Finally, goes back to the head position in which it was before this function

§

Leq()

Compares the next two letters A and B on the main tape, if A <= B, puts 1 in the third position, otherwise puts 0. Finally, goes back to the head position in which it was before this function

§

Gt()

Lt()

§

Eq()

Compares the next two letters A and B on the main tape, if A = B, puts 1 in the third position, otherwise puts 0. Finally, goes back to the head position in which it was before this function

§

Neq()
